# Library-

A Node.js library management system built with Express, MongoDB, and EJS templates.

## About

- Admin and user authentication
- Book management and borrowing requests
- Profile editing and request tracking
- Responsive styled login and registration pages

## GitHub Pages

This repository includes a static GitHub Pages homepage at:

https://eswaran102005.github.io/Library-/

> Note: The app itself is a Node.js server application and must be run locally. GitHub Pages only serves the static homepage.

## Run locally

```bash
npm install
npm start
```

Then open `http://localhost:3000` in your browser.
